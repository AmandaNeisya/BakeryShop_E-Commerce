# BakeryShop_E-Commerce
🍞 Bakery Shop - Sistem E-Commerce Berbasis Web
Sistem E-Commerce Bakery Shop berbasis Web (PHP &amp; MySQL). Project ini dikembangkan sebagai tugas kelompok Sistem ini bertujuan untuk mentransformasi proses pemesanan manual pada toko bakery menjadi sistem berbasis web yang terintegrasi.

📝 Deskripsi Project

Sistem e-commerce ini dirancang untuk mengatasi masalah pencatatan pesanan yang tidak rapi dan sulitnya analisis stok barang. Website ini menyediakan fitur bagi pelanggan untuk mengeksplorasi produk secara interaktif serta menyediakan dashboard admin untuk pengelolaan data produk secara terpusat (CRUD).

🚀 Fitur Utama

Katalog Produk Interaktif: Menampilkan daftar roti dan kue dengan gambar, harga, dan detail produk saat diklik.

Sistem Keranjang Belanja: Memungkinkan pelanggan menampung pesanan dan menghitung total harga otomatis.

Dashboard Admin (CRUD): Akses khusus admin untuk menambah, mengubah, dan menghapus data produk di database.

Integrasi Lokasi: Peta lokasi toko yang terintegrasi dengan Google Maps API.

Autentikasi User: Sistem login menggunakan PHP Session untuk membedakan hak akses Admin dan Customer.

🛠️ Teknologi yang Digunakan

Frontend: HTML5, CSS3, JavaScript.

Backend: PHP.

Database: MySQL.

Design/Tools: VS Code, Git, Canva (untuk aset visual)
